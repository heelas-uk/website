How to set up Hackatime

Step 1: Log in to hackatime.com using your the email associated with your Hack Club account.

Step 1a: If you are receiving the link to your email, don't click the button instead right-click and press copy link

Step 1b: Paste the link into your browser's address bar and press enter.

Step 2: Once logged in, click on your profile icon in the bottom left corner and select "Settings".

Step 3: In the settings menu select "Set up Time Tracking". Note: If you are using Github codespaces you will need to this every time you create a new space

Step 4: Now switch to your code editor of choice if you are in school that will need to be codespaces while out of school you can use whatever your heart desires üòÅ. Search for the code editor you are using and follow the instructions until you enter your API key.

Step 7: Get coding if you upload your code to github you can earn points for the club so we can buy snacks and drinks for our next club meeting
